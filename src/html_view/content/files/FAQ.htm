FAQ - Frequently Asqued Questions

1. ¿Cómo puedo cambiar la escala/ el zoom?

Se tiene cuatro opciones:

  1. La barra izquierda, en la cual se puede aumentar o reducir el zoom según donde se deslice la barra
  2. La Navegación rápida viene con diferenets iconos de zoom
  3. Zoom Box
  4. Doble-clic sobre el mapa

2. ¿Puedo apagar las ortofotos?

Si. Se puede apagar las ortofotos, desactivándolas en la ventana a la derecha. Para abrir la ventana a la izquierda haga clic sobre la flecha café. Se abre la box en las cuales están nombradas las capas.

3. ¿Por qué la aplicación no guarda los cambios cuando modifico los usuarios en la administración de usuarios?

Cuando la aplicación está habilitada para el uso de Active Directory (administración de usuarios de redes Microsoft), sincroniza la información de la base de datos de usuarios de la aplicación con la información de usuarios de la red Microsoft cada vez que un usuario ingresa a la aplicación, perdiendo todos los cambios realizados en la administración de usuarios. Para poder administrar una base de datos de usuarios independientes se debe desactivar el uso de Active Directory.

4. Al activar una capa de información. ¿Cómo puedo visualizar rápidamente el lugar donde se encuentra la información activada?

En la herramienta "Capas", sobre la capa de interés hace clic, aparecerá una ventana con tres tipos de acción, la acción correspondiente es "Acercamiento" y así la aplicación lo llevara donde se encuentra la información.

5. ¿Puedo hacer una consulta en la Herramienta "Capas"?

En la Herramienta "Capas", se busca la información o la capa que se necesita y sobre esta haga clic con el botón derecho y aparecerá un link "consulta". Aparecerá la ventana de consultas.

6. El sistema coordenado tiene un datum predeterminado. ¿Puedo cambiar ese datum para el sistema coordenado UTM?

En primera instancia el servidor se encuentra en coordenadas geográficas (grados). Hacia la parte superior izquierda hay un icono verde llamado "Proyecciones", se presiona la flecha azul que se encuentra a un lado y se desplegaran las proyecciones que están a disposición.

7. Quiero hacer análisis de terreno. ¿Puedo obtener medidas de distancias o áreas en la aplicación?

Una de las herramientas de este servidor de mapas es "Medidas", donde encontramos dos alternativas "Regla" y "área". Una de ellas se utiliza para tomar medidas lineales o distancias, la segunda alternativa sirve para obtener áreas, se dibuja un polígono y se cierra haciendo doble clic posteriormente el valor del área aparece al lado derecho de "Medidas".

Además se puede medir coordenadas de un punto y de un recuadro en el sistema de coordenadas actual. Estos iconos se encuentran en las herramientas en “Consulta”. Los iconos se llaman “Coordenadas Punto” y “Coordenadas Rectángulo”.

8. He dibujado vectores. ¿Cómo puedo limpiar la vista?

Hay dos formas de eliminar los vectores dibujados o también borrar selecciones, en la pestaña de "vectores" se encuentran dos herramientas representadas por dos iconos, el primero de izquierda a derecha sirve para eliminar elementos uno por uno pinchando con el mouse. Los nodos de las poli líneas y Polígonos se tiene que borrar extra. El segundo icono sirve para borrar todo lo de la vista, antes de borrar todo el servidor consulta si desea realizar la acción.

9. A la hora de imprimir. ¿Por qué aparece una opción de "Acercamiento"?

Esta opción existe para que al momento de imprimir se pueda independizar la vista con la impresión. Esto significa que puede cambiar el zoom del mismo lugar para imprimir una región deseada, el zoom se maneja deslizando la barra de derecha e izquierda.

10. Cuando dibujo un polígono. ¿Cómo puedo modificar su forma?

Luego de dibujar un polígono o una línea, si necesita modificar su forma se utiliza la herramienta "Modificar Polígono" que se encuentra en "Vectores", hace clic sobre el icono y posteriormente clic sobre el polígono en ese momento aparecerán los vértices del polígono los cuales puede manejar haciendo clic sobre ellos.

11. ¿Puedo manejar el contraste de los Layers de información con respecto a la base layer y los overlays?

Puede manejar el contraste con la Herramienta "Alpha" que se encuentra en "Mapa", al abrir esta pestaña se encuentra una barra la cual puede deslizar a la derecha para aumentar el contraste y hacia la izquierda para disminuirlo.
